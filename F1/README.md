# Фаза 1: Выбор проектной концепции

## Введение

Данный документ содержит результаты многокритериального анализа проектных альтернатив
с использованием методов **Fuzzy TOPSIS** и **AHP (Метод анализа иерархий)**.
Цель — выбрать наиболее перспективную концепцию для дальнейшей разработки.

---

## Проектные альтернативы

| Код | Проект | Описание |
|-----|--------|----------|
| A1 | **AI Tutor** | Персональный AI-репетитор для обучения |
| A2 | **FoodSaver** | Платформа продажи продуктов с истекающим сроком годности |
| A3 | **LegalBot** | Юридический AI-ассистент для консультаций |
| A4 | **TravelGenie** | AI-планировщик путешествий |

---

## Система критериев

| Код | Критерий | Тип | Пояснение |
|-----|----------|-----|-----------|
| C1 | Социальная значимость | Benefit | Вклад в образование, доступность знаний, польза для школьников и студентов |
| C2 | Экологический эффект | Benefit | Снижение пищевых отходов, поддержка устойчивого потребления |
| C3 | Юридические риски | Cost | Риски ошибок в консультациях, ответственность, регуляторные ограничения |
| C4 | Пользовательская ценность | Benefit | Экономия времени, персонализация маршрутов, удобство для пользователей |
