# Фаза 2 (распределение работ). Проект: **AI Tutor**

Данный документ описывает постановку и решение задачи распределения работ (задач Фазы 3) между **двумя** участниками команды
для проекта **AI Tutor** (персональный AI‑репетитор для обучения).

Критерии распределения:
- каждая задача должна быть назначена **ровно одному** участнику;
- суммарная нагрузка участников должна быть **примерно одинаковой** (баланс по часам);
- распределение должно учитывать **предпочтения** участников (интерес/компетенции) и максимизировать их суммарно.
